# Gemini Deep Research Automation Skill

Use this skill to programmatically delegate comprehensive research tasks or prompts to Gemini Spark with the **Deep Research / Thinking** mode activated by default.

This skill is a wrapper around the core `gemini-spark` automation tool.

## How to Run a Deep Research Query

To interact with Gemini Spark in Deep Research mode, run the script:

```bash
node C:\Users\ibrah\.gemini\config\skills\gemini-deep-research\scripts\index.js [--file "path/to/file"] [--continue <index_or_id>] "Your research query here"
```

### Examples
*   **Start Deep Research (New Chat):**
    ```bash
    node C:\Users\ibrah\.gemini\config\skills\gemini-deep-research\scripts\index.js "Perform a detailed analysis of solar energy efficiency trends over the last decade."
    ```
*   **Continue Deep Research in Existing Chat:**
    ```bash
    node C:\Users\ibrah\.gemini\config\skills\gemini-deep-research\scripts\index.js --continue "Cross-reference the findings with wind energy statistics."
    ```

## Automated Exporter

All Google Workspace files generated by Gemini (Docs, Sheets, Slides) will be automatically downloaded and copied directly into your active working directory.
